# 📌 프로젝트 소개

inSSa는 SSAFY 공지, 일정, 학습 정보 등을 기반으로
AI가 사용자 맞춤형 답변과 일정을 제공하는 서비스입니다.

## ✨ 주요 기능

* 📅 SSAFY 일정 통합 조회
* 🤖 AI 기반 질문 응답
* 🔍 RAG 기반 공지 검색
* 👤 OAuth 로그인
* 🧠 개인 맞춤 추천
